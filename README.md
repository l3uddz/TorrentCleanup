# TorrentCleanup

Sonarr Setup:
-------------

1. Click "Settings" -> "Connect".

2. Add a new "Custom Script".

3. Add the following:

    i. Name: Torrent Cleanup

    ii. On Grab: No

    iii. On Download: Yes

    iv. On Upgrade: Yes

    v. On Rename: No

    vi. Path: /scripts/torrents/TorrentCleanup.py

    vii. Arguments: sonarr

========================================================================
